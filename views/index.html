My Experiences

Research Intern @ IIT Guwahati

Worked on increasing the efficiency of solar cells and to find out their dependence on extrinsic factors such as temperature, soiling and weather, using data science and machine learning, under Dr Prabir Barooah, Department of EEE IIT Guwahati. Collected data of temperature, active power and irradiance by writing various data mining and web scraping algorithms. Decrypted hundreds of encrypted xlsx files to extract data, using python. Wrote efficient python code to render data into usable form, analysed them using python libraries. Deployed a linear regression model to extract temperature coefficient of a PV module. Used several optimisation algorithms to analyse the data, such as YoY method and constraints based minimisation. Manuscript in progress.
